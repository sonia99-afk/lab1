# Анализатор текста

Минимальное веб-приложение для анализа текста, разработанное в рамках лабораторной работы по теме «Понятия “метод” и “методология” в науке и в решении ИТ-задач».

## Функциональность

Приложение позволяет определить:

- количество символов;
- количество слов;
- количество предложений;
- самое длинное слово.

## Выбранная методология

Для разработки использована методология Agile.

Выбор Agile обоснован тем, что приложение небольшое и может разрабатываться поэтапно: сначала создается интерфейс, затем реализуется логика анализа текста, после этого проводится тестирование и исправление ошибок.

## Этапы работы

1. Анализ требований к приложению.
2. Проектирование структуры интерфейса.
3. Реализация HTML-разметки.
4. Оформление интерфейса с помощью CSS.
5. Реализация логики анализа текста на JavaScript.
6. Функциональное тестирование приложения.

## Примененные методы

- анализ требований;
- декомпозиция задачи;
- прототипирование интерфейса;
- алгоритмизация;
- программирование;
- функциональное тестирование.

## Возникшие сложности и способы решения

Основная сложность была связана с корректным выделением слов в тексте. Для решения использовано регулярное выражение, которое учитывает русские и английские буквы, а также цифры.

## Запуск проекта

1. Скачать файлы проекта.
2. Открыть файл `index.html` в браузере./открыть по ссылке
3. Ввести текст и нажать кнопку «Проанализировать».
